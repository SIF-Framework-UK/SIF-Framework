Integration Guide

Connect your applications to the SIF Framework in minutes

Authentication

All API requests require an API key:

Authorization: Bearer sk_prod_your_production_key_here

Get your keys from the Dashboard ‚Üí API Keys & Integration.

Basic Integration Pattern

1. Submit Ethical Query

// JavaScript Example
async function getEthicalRuling(query, context) {
  const response = await fetch('http://your-domain.com/api/ai-adjudicate', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer sk_prod_your_key',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      debate: {
        for: query.proArgument,
        against: query.conArgument
      },
      industry: 'healthcare',
      context: context
    })
  });

  return await response.json();
}

// Usage
const ruling = await getEthicalRuling(
  {
    proArgument: "This action will save lives",
    conArgument: "This violates patient autonomy"
  },
  { urgency: 'high', impact: 'large' }
);

2. Handle Response

// Example response handling
if (ruling.success) {
  console.log(`Decision: ${ruling.ruling.ruling}`);
  console.log(`Confidence: ${ruling.ruling.confidence}`);
  console.log(`Compliance Score: ${ruling.compliance.score}`);

  if (ruling.compliance.compliant) {
    // Proceed with action
    executeApprovedAction();
  } else {
    // Require human review
    escalateForReview(ruling);
  }
}

Language-Specific Examples

Python

import requests

class SIFClient:
    def __init__(self, api_key, base_url="http://localhost:3000"):
        self.api_key = api_key
        self.base_url = base_url

    def adjudicate(self, debate, industry="default"):
        response = requests.post(
            f"{self.base_url}/api/ai-adjudicate",
            headers={
                "Authorization": f"Bearer {self.api_key}",
                "Content-Type": "application/json"
            },
            json={
                "debate": debate,
                "industry": industry
            }
        )
        return response.json()

# Usage
client = SIFClient("sk_prod_your_key")
result = client.adjudicate({
    "for": "Saving lives is ethically mandatory",
    "against": "Autonomy violations are unacceptable"
})

Error Handling

async function safeAdjudicate(debate) {
  try {
    const response = await fetch('/api/ai-adjudicate', {
      method: 'POST',
      headers: { /* ... */ },
      body: JSON.stringify(debate)
    });

    if (!response.ok) {
      throw new Error(`HTTP ${response.status}: ${await response.text()}`);
    }

    const result = await response.json();

    if (!result.success) {
      // Handle API-level errors
      console.error('Adjudication failed:', result.error);
      return getFallbackRuling(debate);
    }

    return result;

  } catch (error) {
    // Handle network errors
    console.error('Network error:', error);
    return getFallbackRuling(debate);
  }
}

Best Practices

  1. Always check compliance scores before taking action
  2. Implement fallback mechanisms for API failures
  3. Cache frequent queries to reduce API calls
  4. Monitor compliance alerts for regulatory changes
  5. Use webhooks for real-time updates
