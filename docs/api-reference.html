API Reference

Complete technical documentation for all SIF Framework endpoints

Base URLs

Development: http://localhost:3000
Production: https://api.sif-framework.com
Staging: https://staging.sif-framework.com

Authentication

API Keys

Authorization: Bearer sk_prod_your_production_key_here

Key Types

Key Type Rate Limit Use Case
sk_prod_* 1000/hr Production applications
sk_test_* 10/hr Testing and development
No key 1/hr Demo and evaluation

Core Endpoints

POST /api/ai-adjudicate

Submit an ethical dilemma for AI adjudication.

Request:

POST /api/ai-adjudicate
Authorization: Bearer sk_prod_abc123...
Content-Type: application/json
{
  "debate": {
    "for": "Argument supporting the action",
    "against": "Argument against the action"
  },
  "industry": "healthcare",
  "context": {
    "urgency": "high",
    "impact_scale": "large",
    "stakeholders": ["patients", "doctors", "public"],
    "time_horizon": "immediate"
  },
  "options": {
    "include_compliance": true,
    "generate_certificate": true,
    "detailed_reasoning": true
  }
}

Response:

{
  "success": true,
  "ruling": {
    "decision": "For",
    "reasoning": "Detailed explanation...",
    "confidence": 0.87,
    "method": "ai_adjudication",
    "principles_applied": [
      {
        "principle": "Beneficence",
        "weight": 1.2,
        "influence": "strong_support"
      }
    ]
  },
  "compliance": {
    "compliant": true,
    "score": 92,
    "breakdown": {
      "GDPR": 95,
      "HIPAA": 89
    },
    "certificate": {
      "id": "COMP-20240115-XYZ789",
      "valid_until": "2025-01-15T10:30:00Z"
    }
  },
  "metadata": {
    "request_id": "req_123456",
    "processing_time": "1.2s",
    "model_version": "2.0.0"
  }
}

GET /api/compliance/score

Get current compliance scores and recommendations.

Request:

GET /api/compliance/score
Authorization: Bearer sk_prod_abc123...

Response:

{
  "overall_score": 92,
  "breakdown": {
    "GDPR": {
      "score": 95,
      "status": "compliant",
      "last_audit": "2024-01-01"
    },
    "HIPAA": {
      "score": 89,
      "status": "compliant",
      "last_audit": "2024-01-01"
    }
  },
  "recommendations": [
    {
      "priority": "high",
      "action": "Enhance documentation for EU AI Act",
      "estimated_effort": "2 weeks",
      "impact": "+15 points on compliance score"
    }
  ]
}

{
  "error": {
    "code": "RATE_LIMIT_EXCEEDED",
    "message": "Rate limit exceeded. Please wait 15 minutes.",

      "limit": 1000,

Code HTTP Status Description
