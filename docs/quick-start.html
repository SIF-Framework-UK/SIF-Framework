Quick Start Guide

Get your SIF Framework instance running in under 5 minutes

Prerequisites

  • Node.js 16+
  • npm or yarn
  • 500MB RAM minimum

Step 1: Installation

# Clone the repository
git clone https://github.com/SIF-Framework-UK/sif-prototype.git
cd sif-prototype

# Install dependencies
npm install

# Start the server
node demo-server.js

Step 2: Verify Installation

Open your browser to:

  • Demo Interface: http://localhost:3000
  • Admin Dashboard: http://localhost:3000/dashboard
